Wanderlust is a web application similar to Airbnb, designed to facilitate the booking and listing of rental properties. The application offers a seamless user experience for both property owners and renters, with features to search for properties, book stays, and manage listings.
Backend: Node.js: To create the server-side application and handle HTTP requests. **Express.js :** As the web application framework for Node.js to build RESTful APIs. MongoDB: As the database to store user data, property listings, and booking information.
User Dashboard: Property owners can manage their listings, view booking requests, and track earnings. Renters can view their booking history and manage upcoming reservations.
The Wanderlust project showcases ability to develop a full-stack web application with a modern tech stack. It demonstrates my skills in frontend and backend development, database management, and creating a user-centric application.
